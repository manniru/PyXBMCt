Detailed Description

Addon UI container with a transparent background.

Control window is displayed on top of XBMC UI,
including video an music visualization!
Minimal example:

class MyAddon(AddonDialogWindow):
    def __init__(self, title='')
        super(MyAddon, self).__init__(title)
        self.setGeometry(400, 300)
        self.setGrid(4, 3)

addon = MyAddon('My Cool Addon')
addon.doModal
del addon

inherited 
Sets the resolution that the coordinates of all controls are defined in.

Allows XBMC to scale control positions and width/heights to whatever resolution
XBMC is currently using.

resolution is one of the following:
    0 - 1080i      (1920x1080)
    1 - 720p       (1280x720)
    2 - 480p 4:3   (720x480)
    3 - 480p 16:9  (720x480)
    4 - NTSC 4:3   (720x480)
    5 - NTSC 16:9  (720x480)
    6 - PAL 4:3    (720x576)
    7 - PAL 16:9   (720x576)
    8 - PAL60 4:3  (720x480)
    9 - PAL60 16:9 (720x480)

Note: default is 720p (1280x720)
Note 2: this is not an actual display resulution. This is the resolution of the coordinate grid
all controls are placed on.
